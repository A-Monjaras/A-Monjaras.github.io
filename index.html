SoyMaistro

SoyMaistro is an innovative web app I developed to assist Latino contractors in creating and sending estimates/bids to clients. Designed for easy adoption, SoyMaistro doesn't require any additional apps or software. Users can access features with a simple text message. By integrating AI-driven solutions, the app further streamlines the estimation and bidding process, enabling contractors to operate more efficiently, expand their reach, and enhance client interactions with minimal effort. The app was built using Node.js and Express, with API integrations to handle messaging and translation services. Data is managed using a PostgreSQL database, and the entire application is deployed on Heroku, ensuring scalability and reliability.
